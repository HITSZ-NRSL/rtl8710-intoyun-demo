# 基于`IntoYun第三方设备SDK`开发的RTL8710实例工程

通过该项目，我们将介绍如何使用`IntoYun第三方设备SDK`开发第三方设备。

## 目录和文件组成

```
+-- components                : 插件文件夹
    +-- intoyun_iot           : IntoYun SDK
    +-- sdk-ameba             : ameba SDK
+-- main                      : 源文件目录
+-- README.md                 : 项目说明文件
+-- project                   : 项目工程文件

```

## 下载工程

```
git clone --recursive https://github.com/IntoYun/rtl8710-intoyun-demo.git
```

## 开发环境搭建

该项目工程使用IAR工具开发。
